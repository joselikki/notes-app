# Installation
<br>

```sh
$ git clone https://github.com/joselikki/notes-app.git  
$ cd notes-app
$ npm install -g .
```

# Usage

Once you installed the notes-app, you can verify it's working by running:

```sh
$ note --version
```

See list of commands:

```sh
$ note --help
```
<br>
